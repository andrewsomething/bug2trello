Bug 2 Trello is a browser extension to add bugs/issues to a Trello board. You probably already know this if you've gotten this far.

It currently supports Launchpad, GitHub, SourceForge, BitBucket, Debian's BTS, and some Bugzilla instances.

Tip When a board is selected, the "Go to Trello" button in the popup will take you directly to that board.

We love feedback and patches!

Bug 2 Trello is open source software. It is released under the MIT license.

The code is available on GitHub, and you can report bugs there.
